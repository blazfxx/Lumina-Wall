# LuminaWall

LuminaWall is a macOS wallpaper app built with SwiftUI. It manages a local library of wallpapers and supports video wallpapers alongside images.

## Features
- Local wallpaper library management
- Video wallpaper playback and looping
- Marketplace view (optional catalog)
- Thumbnail generation and upscaling utilities

## Requirements
- macOS 26
